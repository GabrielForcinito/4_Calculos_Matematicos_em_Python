# 🧮 Cálculos Matemáticos em Python

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aplicar **cálculos matemáticos utilizando Python**, explorando diferentes contextos como geometria, cálculos volumétricos e operações financeiras.

Os exercícios demonstram a evolução na utilização da linguagem e na aplicação da lógica de programação.

## 🛠️ Tecnologias utilizadas

* Python

---

## 🎯 Funcionalidades

* Cálculo de perímetro e área de um retângulo
* Cálculo de volume do tronco de pirâmide
* Cálculo de prestação com aplicação de multa
* Entrada de dados via terminal
* Exibição de resultados formatados

---

## 📝 Projetos incluídos

### 📐 Perímetro e Área do Retângulo

Calcula o perímetro e a área a partir dos valores informados pelo usuário.

### 🔺 Volume do Tronco de Pirâmide

Calcula o volume utilizando fórmula matemática e a biblioteca `math`.

### 💰 Cálculo de Prestação

Calcula o valor total de uma prestação em atraso com base em multa percentual e dias de atraso.

## 💻 Como executar

1. Clone ou baixe este repositório
2. Abra o projeto no **Visual Studio Code**
3. Abra o terminal integrado
4. Execute o arquivo Python com o comando:

```bash id="v7p4zy"
python nome_do_arquivo.py
```

> 💡 Substitua `nome_do_arquivo.py` pelo arquivo que deseja executar.
> 💡 Recomenda-se a execução pelo terminal do VS Code para melhor visualização e interação com o programa.
