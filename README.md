# 🧮 Cálculos Matemáticos em Python

## 📌 Sobre o projeto

Este repositório reúne projetos desenvolvidos em Python com foco na aplicação de cálculos matemáticos e lógica de programação.

Os projetos envolvem diferentes contextos, como geometria, cálculos volumétricos e operações financeiras, demonstrando a evolução na utilização da linguagem.

## 🛠️ Tecnologias utilizadas

* Python

## 🎯 Funcionalidades

* Cálculo de perímetro e área de um retângulo
* Cálculo de volume do tronco de pirâmide
* Cálculo de prestação com aplicação de multa
* Entrada de dados via terminal
* Exibição de resultados formatados

## 📝 Projetos incluídos

### 📐 Perímetro e Área do Retângulo

Calcula o perímetro e a área a partir dos valores informados pelo usuário.

### 🔺 Volume do Tronco de Pirâmide

Calcula o volume utilizando fórmula matemática e a biblioteca `math`.

### 💰 Cálculo de Prestação

Calcula o valor total de uma prestação em atraso com base em multa percentual e dias de atraso.
